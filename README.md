# 漏洞扫描系统 python512

## 项目概述

本漏洞扫描系统基于Python开发，使用轻量级Web框架Flask，搭配MongoDB数据库进行数据存储，同时整合了病毒库工具OpenVAS（GVM）。系统旨在提供全方位的服务器安全解决方案，支持服务器资源监控、端口扫描、漏洞扫描、日志爬取和邮件告警等核心功能。

## 技术栈

- **开发语言**：Python 3
- **Web框架**：Flask
- **数据库**：MongoDB
- **病毒库工具**：OpenVAS（GVM）

## 功能模块

- **服务器资源监控**：实时监控服务器资源使用情况
- **端口扫描**：支持Windows和Linux系统的端口扫描
- **服务器漏洞扫描**：针对Windows和Linux的漏洞扫描
- **日志爬取**：仅支持Linux系统的日志爬取
- **邮件告警**：实时发送安全告警邮件

## 系统角色

- **系统管理员**：负责系统配置和监控
- **安全工程师**：进行漏洞分析和风险评估

## 运行环境

- **操作系统**：支持Windows和Linux
- **虚拟化环境**：兼容VMware

## 部署步骤

1. 配置Python 3环境
2. 安装依赖库Flask和MongoDB
3. 部署OpenVAS（GVM）病毒库工具
4. 部署Web应用
5. 运行系统初始化脚本
6. 开始监控和管理
